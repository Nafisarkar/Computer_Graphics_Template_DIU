Steps:

    Create a C++ Empty Project:
        Open Visual Studio and create a new project.
        Select "Empty Project" under "C++" templates.

    Extract GLEW and FreeGLUT:
        Extract the contents of the GLEW and FreeGLUT downloads to a location on your computer.

    Configure Project Properties:
        Right-click on the solution in Solution Explorer and select "Properties".

    Set Additional Include Directories:
        Navigate to C/C++ -> General -> Additional Include Directories.
        Add the paths to the include folders of GLEW and FreeGLUT.

    Link Libraries:
        Navigate to Linker -> Input -> Additional Dependencies.
        Add glew32.lib and freeglut.lib.

    Set Additional Library Directories:
        Navigate to Linker -> General -> Additional Library Directories.
        Add the paths to the lib folders of GLEW and FreeGLUT.

    Copy DLLs:
        Open the project folder in File Explorer through Visual Studio.
        Copy the glew32.dll and freeglut.dll from the GLEW and FreeGLUT folders to your project directory.
        Additionally, copy these DLLs to C:\Windows\System32.

    Include Headers in Your Code:

cpp
#include <Windows.h>
#include <GL/glew.h>
#include <GL/freeglut.h>
#include <iostream>
